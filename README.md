# Odak Patoloji

[Odak Patoloji & Sitoloji Laboratuvarı](https://www.odakpatoloji.com.tr) resmi web sitesi projesidir.

Bu proje, modern web teknolojileri kullanılarak geliştirilmiş, hastalar ve doktorlar için patoloji laboratuvarı hizmetlerini, testlerini ve kurumsal bilgilerini sunan bir platformdur.

## Proje Hakkında

Odak Patoloji, Salihli'de hizmet veren köklü bir patoloji laboratuvarıdır. Bu web sitesi aracılığıyla:
- Laboratuvarın tarihçesi ve ekibi tanıtılmakta,
- Yapılan testler ve kullanılan cihazlar hakkında bilgi verilmekte,
- Güncel haberler ve duyurular paylaşılmaktadır.

Web sitesine [https://www.odakpatoloji.com.tr](https://www.odakpatoloji.com.tr) adresinden ulaşabilirsiniz.

## Teknolojiler

Proje aşağıdaki teknolojiler kullanılarak geliştirilmiştir:

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Dil:** [TypeScript](https://www.typescriptlang.org/)
- **UI Kütüphanesi:** [React 19](https://react.dev/)
- **Stil:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animasyon:** [Motion](https://motion.dev/) & [Tailwind Animate](https://github.com/jamiebuilds/tailwindcss-animate)
- **İkonlar:** [Lucide React](https://lucide.dev/)
- **Fontlar:** Plus Jakarta Sans, Lora, IBM Plex Mono

## Kurulum ve Geliştirme

Projeyi yerel ortamınızda çalıştırmak için aşağıdaki adımları izleyebilirsiniz:

1. Bağımlılıkları yükleyin:

```bash
npm install
# veya
yarn install
# veya
pnpm install
# veya
bun install
```

2. Geliştirme sunucusunu başlatın:

```bash
npm run dev
# veya
yarn dev
# veya
pnpm dev
# veya
bun dev
```

3. Tarayıcınızda [http://localhost:3000](http://localhost:3000) adresini açarak uygulamayı görüntüleyebilirsiniz.

## Dağıtım (Deployment)

Bu proje [Vercel](https://vercel.com) üzerinde barındırılmaktadır ve Next.js'in sunduğu optimizasyonlardan faydalanmaktadır.

Daha fazla bilgi için [Next.js Dağıtım Dokümantasyonu](https://nextjs.org/docs/app/building-your-application/deploying) incelenebilir.
